Prism.js & Reveal.js

This example shows how to using prism.js code highlighting in a reveal.js presentation.

SPARQL example


	SELECT ?class (COUNT(DISTINCT ?p) as ?NoP)

	WHERE {
	?s a ?class ;
	?p ?o .
	FILTER STRSTARTS(STR(?class), "http://data.europa.eu/a4g/ontology#")
	}

	ORDER BY DESC(?NoP)
	LIMIT 10
